# Churrería Carmelo Web Project

## Description
This project is a web application for **Churrería Carmelo**, a traditional churreria in Las Palmas de Gran Canaria. The website aims to showcase the menu, the history of the establishment, and provide contact information for customers. It features a responsive design, interactive maps, and a modern user interface.

## Third-Party Components
To enhance the functionality and user experience, the following third-party components and libraries have been used:
- **React Router DOM**: For handling navigation between pages (Home, Carta, Contacto).
- **Leaflet & React Leaflet**: To display an interactive map showing the location of the churreria.
- **Vite**: As the build tool and development server for a fast developer experience.
- **ESLint**: For maintaining code quality and checking for errors.

## Other Sections

### Project Structure
The project follows a clean architecture with:
- `src/components`: Reusable UI components like Header, Footer, and SpecialtyCard.
- `src/pages`: Main view components for each route (Home, Carta, Contacto).
- `src/data`: Static data files (e.g., menu data).

### Features
- **Responsive Design**: Adapts to mobile, tablet, and desktop screens using Media Queries and Flexbox.
- **Dynamic Content**: Menu items and specialties are rendered from JSON arrays.
- **Reusable Components**: Buttons, cards, and layout elements are modularized.

### Installation
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local server.
